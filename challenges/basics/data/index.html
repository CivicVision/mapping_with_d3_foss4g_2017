Basics

Adding or removing elements

d3.selectAll('h1').data(1,2,3])
d3.selectAll('h1').data(1,2,3]).enter()
d3.selectAll('h1').data(1,2,3]).exit()
d3.selectAll('h1').data(1,2,3]).enter().append('h1')
d3.selectAll('h1').data(1,2,3]).exit().remove()

This is the toughest part to grasp. But it is very important one for d3.

The concept is as follows:

  • You select elements you want to create/update
  • You join the data with these elements
  • You change exisitng elements
  • Enter the selection to create new elements
  • Merge for updating both
  • Exit for removing elements

Excercise 1

Create a list with names from the data.

Challenge Show solution 

var lis = d3.select('ul.names').selectAll('li').data(data);
lis.enter().append('li')
.merge(lis).html(function(d) {
  return ("Name: "+d.name+" ("+d.gender+"), "+d.age); 
});

Excercise 2

Create three circles which don't overlap. Fill the existing one with green and the rest red.

Help:

r is the radius, cx is the x coordinate from the center and cy is the y coordinate from the center
https://www.w3schools.com/graphics/svg_circle.asp

Challenge Show solution 
var circles = d3.select('svg').selectAll('circle').data(data);
circles.attr('fill', 'green');
circles.enter().append('circle').attr('fill', 'red')
.merge(circles)
.attr('cx', function(d,i) { return d;})
.attr('cy', function(d,i) { return 10*(i+1);})
.attr('r', 5);
