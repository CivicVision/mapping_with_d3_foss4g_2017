Basics

Changing attributes


d3.select('body').attr('name', valueOrFunction)
d3.select('body').style('css', valueOrFunction)
d3.select('body').text(yourTextOrFunction)
d3.select('body').html(yourHtmlOrFunction)
You can either use a value or you can use a function to change things. The functions will come handy once we join the elements with data.

Example

d3.select('.test').attr('id', 'my-first-test')
d3.select('.test').attr('id', function(d) { return 'my-first-test';})
d3.selectAll('.test').attr('id', function(d,i) { return 'my-id'+i;})
